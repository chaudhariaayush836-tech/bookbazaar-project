# BookBazaar – Online Book Store

## 📌 Overview
BookBazaar is an eCommerce project built using Odoo to sell popular self-help books at affordable prices.  
The goal is to make reading accessible for students and beginners.

---

## 🚀 Features
- 📚 Product listings (Atomic Habits, Ikigai, Psychology of Money, etc.)
- 💸 Discount system (₹100 OFF on orders above ₹299)
- 💳 Razorpay payment integration
- 🚚 Shiprocket delivery integration
- 📲 Instagram-based marketing strategy

---

## 🛠 Tech Stack
- Odoo (Website & eCommerce)
- Razorpay (Payments)
- Shiprocket (Logistics)

---

## 🎯 Goal
